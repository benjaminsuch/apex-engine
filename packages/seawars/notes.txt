Anforderungen für das "build" Script

devdeps:
- esbuild

Sonstiges:
- Der vom User genutzte Package Manager muss in "packageManager" (package.json) hinterlegt werden
- Die in "scripts" ausgeführten Scripts müssen dem "packageManager" angepasst werden

Build:
- "buildTypes" wird nicht notwendig sein, da es sich beim Spiel um keine Library, sondern einem "Endprodukt" handelt. Das Spiel wird also nicht woanders importiert, so dass die generierten Types von Nutzen wären.
- esbuild unterstützt kein HMR (siehe https://github.com/evanw/esbuild/issues/645#issuecomment-755215007).
- Es muss vermutlich ein eigenes Build-System a la Vite gebaut werden.
